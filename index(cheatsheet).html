CSS selectors and specificity

Class Selectors

Tag Name Description
<Type> Type The CSS type selector matches elements by node name. In other words, it selects all elements of the given type within a document.
<Class> Class The CSS class selector matches elements based on the contents of their class attribute.
<ID> ID The CSS ID selector matches an element based on the value of the element's id attribute. In order for the element to be selected, its id attribute must match exactly the value given in the selector.

Attributes

Selector Type Name Description
<Color> Color Changes the color of the text within the given attribute
<Text Transform> Text Transform specifies how to capitalize an element's text.
